# Akula Clothing Store — Showcase UI

A fast, zero-backend site to showcase items (no e‑commerce). Built with **Vite + React + Tailwind**.

## 🚀 Quickstart
```bash
npm install
npm run dev
# open http://localhost:5173
```

## 🧱 Structure
```
src/
  components/        # UI parts (Hero, Gallery, Footer, etc.)
  data/items.ts      # Your catalog data (replace with your items)
  App.tsx            # Page composition
```

## 🖼 Update your items
Edit `src/data/items.ts` with real names, prices, categories, and image URLs.

## 🌐 Deploy (Vercel or Netlify)
- Push to GitHub (main branch).
- Import the repo in Vercel (or Netlify) and deploy with defaults.

## 🔧 Scripts
- `npm run dev` - Start local dev server
- `npm run build` - Production build
- `npm run preview` - Preview build locally
```
